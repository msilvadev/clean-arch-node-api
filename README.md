   ```sh
   git clone https://github.com/msilvadev/clean-arch-node-api.git
   ```

   ```sh
   yarn
   ```

#

> ## ðŸ”¥ Running the app

### 1. If you have docker and docker-compose installed you can run:

- Up the container
  ```sh
  yarn up
  ```
- Down the container
  ```sh
  yarn down
  ```

### 2. If you don't have docker

```sh
yarn build && yarn start
```

- Have a MongoDB server running
